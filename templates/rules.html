How to Play CODE

Objective

Be the first player to crack your opponent's secret code by deducing the correct 4-digit combination.

Setup

  • Each player receives a random 4-digit code card
  • Players draw 5 cards from the deck to start
  • The deck contains number cards (0-9 in 4 colors) and action cards

Gameplay

On Your Turn:

  1. Play a Card: Play a number or action card from your hand
  2. Make a Guess: Try to guess one digit of your opponent's code
  3. Get Feedback:
    • ✓ Correct digit in correct position
    • ~ Correct digit in wrong position
    • ✗ Digit not in code
  4. Draw a Card: Draw from the deck to replenish your hand

Action Cards

  • JOKER: Wild card, can be played as any number
  • AUSSETZEN (Skip): Opponent skips their next turn
  • PLUS2: Opponent draws 2 extra cards
  • TAUSCH (Swap): Swap hands with opponent
  • GESCHENK (Gift): Give a card to opponent
  • RESET: All players draw new codes
  • RICHTUNGSWECHSEL (Reverse): Reverse play direction (multiplayer)

Winning

The first player to correctly guess all 4 digits of their opponent's code in the correct positions wins the game!
